# Summer Intern Challenge 2020
Intern AI/ML/IoT/Robot Challenge Learn and demonstrate how to manage and deploy robot applications to robots using AWS RoboMaker, ROS, AWS IoT and Nvidia Jetbot.  We will be exploring machine learning, monitoring, and analytics that enable a robot to stream data, navigate, communicate, comprehend, and learn.
